# Roadmap и планируемые доработки

Эта директория предназначена для проектных заметок, roadmap-документов и вариантов будущих доработок, которые пока не являются активной реализацией.

Документы здесь помогают сохранять технические идеи до момента, когда они будут превращены в полноценную задачу, дизайн или PR.

## Правила использования

- Описывайте здесь только планируемые или исследовательские изменения.
- Не считайте документы из этой директории текущим runtime-контрактом системы.
- Если идея переходит в реализацию, обновляйте соответствующие документы в `docs/technical/` и `docs/guides/`.
- Если план устарел, помечайте это в документе вместо тихого удаления контекста.
